# Solar Energy Forecasting

This repository contains the Python implementation for a solar energy forecasting project using time series regression and machine learning techniques.

## Contents
- Jupyter Notebook implementing data cleaning, EDA, feature selection, model training, and evaluation
- Linear Regression and Random Forest Regression models
- January 2026 solar energy production forecast

## How to use
Download the notebook and run it in Jupyter Notebook or Google Colab.
